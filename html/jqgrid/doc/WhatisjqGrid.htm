jqGrid is a Ajax-enabled JavaScript control that provides solution for representing and manipulating tabular data on the web. Since the grid is client-side solution and loading data dynamically through Ajax callbacks, it can be integrated with any server side technology – like PHP, ASP, Java Servlets,  JSP, ColdFusion, Perl.

jqGrid  use a jQuery Java Script Library and is written as plugin. For more information on what jQuery is, please refer to jQuery web site.

I started the idea when I need easy way to represent database information in my project. The first requirement was a speed and the second independence from server side technology and database backend.
