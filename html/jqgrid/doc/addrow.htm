This method can be used to add data to the server.

Description
This method uses colModel and editurl parameters from jqGrid
Calling: jQuery("#grid_id").editGridRow( "new", options );

The options are the same as these in Edit row

Note: additionally to the values that are posted to the server are added two parameters

id=_empty and oper=add to determine at server that the operation is insert
