Calling conventions:

jQuery("#table_id").jqGridMethod( parameter1,...parameterN )

Where
table_id is the id of the already constructed jqGrid.
jqGridMethod is a method applied to this jqGrid.
parameter1,...parameterN - a list of parameters

IMPORTANT NOTE: The colored methods will be removed in the next release. Instead you can use setGridParam and getGirdParam methods. Read the replacement below for every method. 

Below is a list of available methods
  
Method
Description
getGridParam
Parameters:
name - string
This method return the value of the requested parameter. The name is the name from options array. For a particular name see Options. If the name is not set the entry options are returned.
setGridParam
Parameters
object - array
set a particular parameter. Note - for some parameters to take effect a trigger("reloadGrid") should be executed. Note that with this method we can override events like onSelectRow and etc.  Example:

setGridParam({
url:"newurl",
page:1,
onSelectRow:function(id){/*here is the new code*/}
});
getUrl
getGridParam("url")
Parameters: none
Return the current url from options array
getSortName
getGridParam("sortname")
Parameters: none
Return the name of last sorted column
getSortOrder
getGridParam("sortorder")
Parameters: none
Return the last sorted order
getSelectedRow
getGridParam("selrow")
Parameters: none
Return the id of the selected row, null if row is not selected
getPage
getGridParam("page")
Parameters: none
Return the current page number.
getRowNum
getGridParam("rowNum")
Parameters: none
Return the current number of requested rows
getDataType
getGridParam("datatype")
Parameters: none
Return the current datatype.
getRecords
getGridParam("records")
Parameters: none
Return the current number of records in grid.
getDataIDs
Parameters: none
Returns: array
Return array of the id's in the current grid view. Empty array if no data is available.
getMultiRow
getGridParam("selarrrow")
Parameters: none
Returns: array
Return array of id's of the selected rows when multiselect options is true. Empty array if not selection.
setUrl
setGridParam({url:newvalue})
Parameters:
url - string
Set a new url, replacing the older.
setSortName
Parameters:
sortname - string
Set a new sort name
setSortOrder
setGridParam({sortname:newvalue})
Parameters:
sortorder - string (asc or desc)
Set a new sort order
setPage
setGridParam({page:newvalue})
Parameters:
page - integer >0
Set a new page number
setRowNum
setGridParam({rowNum:newvalue})
 
Parameters:
rownum - integer > 0
Swwwet a new number of requested rows.
setDataType
setGridParam({datatype:newvalue})
Parameters:
datatype - string (xml,json.xmlstring,jsonstring, clientSide)
Set a new datatype.
setSelection
Parameters:
rowid - string
Toggle a selection of the row with id = rowid
resetSelection
Parameters: none
Reset (unselect) the selected row(s). Also works in multiselect mode.
getRowData
Parameters:
rowid - string
Return - array
Return array with data of the requested id = rowid. The returned array is of type name:value, where the name is a name from colModel and the value is a actual value.
Return empty array if the row can not be found.
delRowData
Parameters:
rowid - string
Return - boolean
Deletes row with the id = rowid. Return true is the operation success, otherwise false. This operation does not delete a data from the server.
setRowData
Parameters:
rowid - string
data - array
Return: boolean
Updates the values (using data array) in the row with the given rowid. The syntax of data array is: {name1:value1,name2: value2…} where the name is the name of the column as described in the colModel and the value is the new value. Return true on success, false otherwise.
addRowData
Parameters:
rowid - string
data - array
position - string (first, last) - default last
Insert new row with id rowid at the position  using data array.The syntax of data array is: {name1:value1,name2: value2…} where the name is the name of the column as described in the colModel and the value is the value. Return true on success, false otherwise
hideCol
Parameters:
colname - string
Hide a column with a given colname. The colname must be a valid name from colModel. The width of the grid does not change.
showCol
Parameter:
colname - string
Show a column with a given colname. The colname must be a valid name from colModel. The width of the grid does not change.
setCaption
Parameters
caption - string
Set a new caption of the grid. If the Caption layer was hidden it is shown.
.trigger("reloadGrid");
Parameter: none
This reload the grid with the current settings. This mean that a new request is send to the server if datatype is xml or json.
This method should be applied to already constructed grid - i.e. jQuery("#mytable_id").trigger("reloadGrid");
