Editing Methods
 Previous Top Next
We have a three methods to manipulate the data.

editRow( rowid, keys, oneditfunc, succesfunc, url, extraparam, aftersavefunc )

where

rowid is the id of the row to edit
keys - when set to true we can use Enter key to save the row and Esc to cancel editing.
oneditfunc - fires after succes editing of row. Parameter passed to this function is the id of the row.
Parameters succesfunc, url, extraparam, aftersavefunc have sense only when keys parameter is set to true. These parameters are passed to saveRow method when a Enter key is pressed. For more information see saveRow method below.


When this method is called on particular row we actually read the data for the editable fields and construct the appropriate elements defined in edittype and editoption.
This method can be called only on already constructed grid.


saveRow (rowid, succesfunc, url, extraparam, aftersavefunc)

where

rowid is the id of the row to save
succesfunc if defined this function is called immediately after the request is successfully. To this function is passed the data returned from the server. Depending on the data from server this function should return true or false.
url when defined this parameter replaces the editurl parameter from options array.
extraparam is array of type name:value. When set these values are posted again with the oter values to the server.
aftersavefunc  - when defined this function is called after the data is saved to the server. Parameter passed to this function is the rowid and the result from the request.

When this method is called to particular row the data is POSTED to the server in format name:value, where the name is a name from colModel and the value is the new value.
Additionally to the posted data is added a pair id=rowid.

This method can be called only on already constructed grid.


restoreRow(rowid)

where

rowid is the row to restore

This method restores the data to original values before the editing of the row.

This method can be called only on already constructed grid.
