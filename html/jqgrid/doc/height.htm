Height

The height of the grid can be controlled via height and rowheight parameters in options array.

The default value for height is 150px. The default value for rowheight is null, which means that this parameter is not used. I recommend not to use rowheight parameter, since there is a more elegant way to do that what rowheight do. The rowheight parameter was created with the sence to use a grid with a float height. That means - if we have 10 rows - the height of grid will be 10*rowheight. Next if the requested rows are 5 the height of grid will be 5*rowheight. Setting this parameter is a little difficult, since there is a need to perform some testing.

So I will speak only for height parameter.

The height parameter can be set in pixel or percent or any valid height measure.
The default settings is in pixel. To set the grid height in 200 pixel we need to set only the number - i.e.

height : 200

A very important thing here is that if we set the height of the grid this is the height
only of the Body layer. The whole height of grid is - the height in option array + the
height of the Header layer + height of the Caption layer (if present) + height of the
pager (if set as a part of grid).

Since we can set the height in different measures we can use the trick with the percentage. Setting the height to 100% means the same as setting the rowheight parameter.
