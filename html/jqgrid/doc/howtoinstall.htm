How to install
 Previous Top Next
First you need to download the jQuery JavaScript library. This library can be downloaded from www.jquery.com. Please download the latest stable version of jQuery library and not a development version.

You need to download the jqGrid plugin from http://trirand.com/jqgrid/jqGrid.zip

I suppose that you are familiar with some thinks like web server, database server and scripting programming language. I suppose too that you have installed this software.

Create a directory on your web server, so that you can access it: http://myserver/mydir/ , where mydir is the name that you have created.

Place the jQuery library in that directory; unpack the jqGrid.zip in the same directory. You should have this directory structure:

-         jquery.js;
-         jquery.jqGrid.js;
-         js
-         - grid.basic.js
-         - grid.inlinedit.js
-         - grid.subgrid.js
-         - grid.formedit.js
-           - jqModal.js
-           - jqDnR.js
-         themes
-         - basic
-         - coffee
-         - green
-         - sand
-         - jqModal.css

where:

jquery.js is the jQuery library,

jquery.jqGrid.js a main module for including different plugins depending on the needs.
grid.basic.js is the main plugin. Without this plugin, all other plugins are unusable.
grid.inlinedit.js plugin is used if we want to have inline editing.
grid.subgrid.js plugin is used if we want to have so named subgrid
grid.formedit.js plugin is used for form editing, adding and deleting data.
grid.postext.js plugin  is used to manipulate post Data array
jqModal.js plugin - should be included if you want to use form editing (modal dialogs)
jqDnR.js plugin - should be included if you want to use form editing (drag and resize)

themes directory that contain different styles for the grid.

If  you want to use all of the features of jqGrid you do not need to do anything.

If you want to use some of the features and/or only the basic functionalities of jqGrid you need to edit jquery.jqGrid.js file and configure it depending on the needs. This file must be edited in case you place the javascript files to other place. This file is simple and can be easily configured.

function jqGridInclude()
{
    var pathtojsfiles = "js/"; // need to be ajusted
    // if you do not want some module to be included
    // set include to false.
    // by default all modules are included.
    var minver = true;
    var modules = [
        { include: true, incfile:'grid.base.js',minfile: 'min/grid.base-min.js'}, // jqGrid base
        { include: true, incfile:'grid.formedit.js',minfile: 'min/grid.formedit-min.js' }, // jqGrid Form editing
        { include: true, incfile:'grid.inlinedit.js',minfile: 'min/grid.inlinedit-min.js' }, // jqGrid inline editing
        { include: true, incfile:'grid.subgrid.js',minfile: 'min/grid.subgrid-min.js'} //jqGrid subgrid
   ];
    for(var i=0;i<modules.length; i++)
    {
        if(modules[i].include == true) {
            if (minver != true) IncludeJavaScript(pathtojsfiles+modules[i].incfile);
            else IncludeJavaScript(pathtojsfiles+modules[i].minfile);
        }
    }
    function IncludeJavaScript(jsFile)
    {
      document.write('<script type="text/javascript" src="'
        + jsFile + '"></script>');
    }
}
jqGridInclude();

If you have different path to javascript files you must change the value of the variable pathtojsfiles in the appropriate way.
If you want to exclude some module from loading you simple must change the value of the include name from true to false which is a part from modules array variable.
If you plan to use form editing module you should include the jqModal.js, jqDnR.js and jqModal.css files in your html page.
Now you are ready to use a jqGrid.
