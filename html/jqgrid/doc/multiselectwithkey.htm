Multiselect with key
 Previous Top Next
Another feature of jqGrid is that we can define a key to make a multiselection. In other words the selection is done when the user hold down the defined key. To achieve this we should use multikey option in options array. This can be done like this:
multikey : "altKey"
In this case the multiselection is done only when the user hold down the "Alt" key.
The possible values can be : "shiftKey","altKey","ctrlKey"
