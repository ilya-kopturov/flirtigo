This is original idea and implementation from Paul Tiseo.

Module name: grid.postext.js

Description: The main purpose of this module is to manipulate the parameters to url via array and get a user defined data from the response. For user defined data please refer to Data Types.

Installation: To enable this module you should enable it in jquery.jqGrid.js.

Manipulating parameters:
In the option array of the grid is added new option postData. By default this is empty array.
The values of this array are added via $.extend to the ajax request. To manipulate the values of the array we can use these methods:

$("#mygrid").getPostData() return all parameters passed to the grid url. The returned value is array of type name:value.

$("#mygrid").setPostData( newdata)  set a new set of parameters overriding the existing ones.
newdata should be array of type name:value. Example {myparam:"myvalue"}
Note that the page, rowNum, sortorder, sortname parameters are not changed. To change these use setGridParam method.

$("#mygrid").appendPostData( newdata) replace or append new parameters to the array. newdata should be array of type name;value

$("#mygrid").setPostDataItem( Key, Val) set new or replace the value of the existing item in the array. Key is the name and Val is the value of the item.

$("#mygrid").getPostDataItem( key)  return the value of the requested item with name key

$("#mygrid").removePostDataItem( key) deletes a specified item with name = key from the array.
