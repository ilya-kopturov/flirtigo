This is not easy answer. Instead that the primary goal of jqGrid is to read database information using Ajax it can be used with success to read local (array) data. Below is a little guide for different situations.

In all situations you need a jQuery Java Script library. The version of jQuery should be greater or equal than 1.1.4.

In case if you plan to manipulate and represent local data – i.e. array data,  data stored in xml file,  data stored in JSON file ( I mean that this data is static and not generated from any server side technology and database) you need only a jQuery library, jqGrid plugin and web browser.

In case if you plan to manipulate and represent a database data over web you need - jQuery library, jqGrid plugin, web server (Apache, Tomcat) , database backend (Postgre SQL, Oracle, MSSQL, MySQL..), server-side scripting language (PHP, ASP…) and web browser.
