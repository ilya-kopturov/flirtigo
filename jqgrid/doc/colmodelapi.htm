colModel API
 Previous Top Next
Syntax:

colModel : [ {name:'name1', index:'index1'...},{...}...]
  
Option
Description
name
Type: string
Set the unique name in the grid for the column. This option must be set.
index
Type: string
Set the index name when sorting. Passed as sidx parameter.
Default value: the order of cell
width
Type: number
Set the initial width of the column in pixels.
Default value: 150 pixels
align
Type: string
Defines the alignment of the cell in the Body layer, not in header cell. Possible values - left, center, right.
Default value: left
key
Type: boolean
In case if there is no id from server this can be set as as id for the unique row id. Only one column can have this property. If there are more than one key the grid finds the first one and the second is ignored.
Default value: false
hidden
Type: boolen
Defines if this column is hidden at initialization.
Default value : false
resizable
Type: boolean
Defines if the column can be risized
Default value: true
sortable
Type: boolean
Defines is this can be sorted.
Default value: true
xmlmap
Type: string
Defines the xml mapping for the column in the incomming xml file. Use a CCS specification for this
Default value: none
jsonmap
Type: string
Defines the json mapping for the column in the incomming json string.
Default value: none
sorttype
Type: string
Used when datatype is clientSide. Defines the type of the column for appropriate sorting. Possible values:
int - for sorting integer
float - for sorting decimal numbers
date - for sorting date
text - for text sorting
Default value: text
datefmt
Type: string
Only in combination with sorttype:date. Determines the expected date format for that column. Uses a PHP like date formatting.
Default value: ISO Date - Y-m-d
editable
Type: boolean
Defines if the field is editable. This option is used in inline and form modules.
Default value: false
edittype
Type: string
Defines the edit type for inline and form editing
Possible values: text, textarea, select, checkbox
See Inline editing and form editing
Default value: text
editoptions
Type: array
Array of allowed options (attributes) for edittype option
Default value: empty array
