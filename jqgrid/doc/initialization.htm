Inline editing is a quick way to update database information. To do that we need to enable this feature. For more information refer to How to install

Related methods: editRow, saveRow, restoreRow
Related options in colModel: editable, edittype, editoptions.
Related options in option array: editurl

First what we should do is to configure the colModel options.
By default all columns are not editable. Below is a description of these options.

editable defines if the field is editable. To enable this set the value too true.
edittype  defines the type of of the editable field. Possible values: text, textarea, select, checkbox. The default value is text.
editoptions Array of allowed options (attributes) for edittype option.

Description of edit types:

text - when we define that the field is of type text we construct a input tag of type text:

<input type="text" ...../>

In edit option we can set all possible attributes for this filed.
Example
editoptions : {size:10, maxlength: 15} will construct a input

<input type="text" size="10" maxlength="15" />

Additionally to the these setting a some additional attributes are added:
id - the id that is added to this element is a combination of the id of the row and the name - rowid_name
name - the name from colModel
value - the contents of the cell.

Consider the example above and suppose that the id of the row is 12 and the name is invdate then the result is:

<input type="text" id="12_invdate" name="invdate" size="10" maxlength="15" value="someval"/>


textarea defines a input type textarea

<input type="textarea" .../>

Use editoptions to add additional attributes to this type. To these attributes we add additionally
id and name attributes as these in text type.

checkbox defines a input type checkbox

<input type="checkbox" .../>

use edit options to define the checked and unchecked value. The first value is checked.
Example editoptions: { value:"Yes:No" } defines a checkbox in which when the value is Yes the checkbox becomes checked, otherwise unchecked. This value is passed as parameter to the editurl.

To this element we add additionally id and name attributes as these in text type.

select defines a single select box of type

<select>
<option value='val1'> Value1 </option>
<option value='val2'> Value2 </option>
...
<option value='valn'> ValueN </option>
</select>

to construct this element we should use a editoptions array with the element value. Every pair value text is separated with : . Every option is separated with ;

Example

editoption: { value:"FE:FedEx;IN:InTime;TN:TNT"} will construct

<select>
<option value='FE'> FedEx </option>
<option value='IN'> InTime </option>
<option value='TN'> TNT </option>
</select>

To this element we add additionally id and name attributes as these in text type.
