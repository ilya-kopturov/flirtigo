Simple multiselect
 Previous Top Next
Multi selection is a way to select more than one row in grid and after that to perform some actions with the selected rows. jqGrid allow a more than one way of multi selection.
Related options in options array
multiselect
multikey
Related methods:  getMultiRow, getSelectedRow


The default value of multiselect is false, which disables this feature by default. When multiselect is set to true a additional column is added at left side of the grid. This row add additional 28 px to the grid width.

When the grid is constructed the content of this column if filled with a check box element. When we select some row the check box become a state checked. When we select another row the previous row does not change the state. When we click on a row that is selected, the state becomes unchecked and the row is unselected.


As seen on the figure at a header we have a common check box. When we check this box all the rows will be selected. When we unchecked this box all the rows are unchecked.
To obtain selected rows we can use getMultiRow method. Using our example we can write this

jQuery("#list").getMultiRow(); ==> ["11","9"]

This command return array with the selected rows. The values in array are the id's of the selected rows.

The line

jQuery("#list").getSelectedRow();
will return the id of the last selected row. This is only a scalar value.
